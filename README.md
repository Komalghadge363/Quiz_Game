An interactive multiple-choice quiz game built with a **frontend-backend architecture** that presents questions, collects user responses, evaluates answers, and displays the final score.

The application supports **multiple question types**, showcasing strong fundamentals in **front-end UI handling**, **backend logic**, and **API-based data flow**.
